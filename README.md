# 🌾 Digi-Agri AI: Digital Agriculture Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Status](https://img.shields.io/badge/Status-Planning%20Phase-blue)]()
[![Development](https://img.shields.io/badge/Development-16%20Week%20MVP-green)]()

AI-powered mobile app empowering small-scale farmers in developing regions with personalized agricultural intelligence.

## 🎯 Mission

Increase crop yields by **30%**, reduce input costs by **20%**, and promote sustainable farming practices for 1,000+ small-scale farmers in South Sudan and East Africa.

## ✨ Features

| Feature | Description | Status |
|---------|-------------|--------|
| 🌱 **Fertilizer Advisor** | AI-powered NPK optimization based on soil & weather | Week 3 |
| 🐛 **Pest Identifier** | Computer vision disease detection (92%+ accuracy) | Week 5 |
| 💬 **AI Chat Assistant** | 24/7 multilingual support (English, Arabic, Dinka) | Week 6 |
| 📊 **Crop Progress Tracker** | Yield predictions & growth stage alerts | Week 3 |
| 🏪 **Marketplace** | Local supplier directory & price comparison | Week 4 |
| 📱 **Offline Mode** | Works without internet; syncs when connected | Week 4 |
| 🌍 **Multilingual** | Support for English, Arabic, Dinka, & more | Week 6 |
| 🎤 **Voice I/O** | Voice input/output for low-literacy users | Week 6 |

## 🏗️ Tech Stack

```
Frontend:       Flutter (iOS/Android) + Riverpod
Backend:        Firebase + Cloud Functions
AI/ML:          TensorFlow Lite + Hugging Face + Llama 2
Database:       Firestore + SQLite (offline)
Storage:        Firebase Cloud Storage
Languages:      Dart, TypeScript, Python
```

## 📋 Quick Start

### Prerequisites
- Flutter 3.x+
- Python 3.8+
- Git
- Firebase account
- Hugging Face API token (for AI features)

### Installation

```bash
# Clone repository
git clone https://github.com/Maror-dev/digi-agri-ai.git
cd digi-agri-ai

# Run setup script
bash scripts/setup.sh

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Start development
cd mobile_app
flutter run
```

## 🌍 Impact

**The Problem:**
- Small-scale farmers in South Sudan face:
  - High input costs (fertilizers, seeds, pesticides)
  - Crop diseases causing yield losses
  - Limited access to expert agricultural advice
  - Unpredictable yields due to climate variability

**Our Solution:**
Digi-Agri AI provides personalized, AI-driven agricultural intelligence accessible to farmers with basic smartphones and offline capability.

**Expected Impact (Year 1):**
- 1,000+ farmers reached
- 30% average yield increase
- $200,000+ in cumulative farmer income increase
- 20% reduction in input costs
- Sustainable farming practices adoption
